# Changelog

Все значимые изменения в проекте будут документироваться в этом файле.

Формат основан на [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
и проект следует [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- Сохранение пресетов размеров входа в localStorage — пользовательские пресеты сохраняются между сессиями (ключ `risk-calculator-presets`).

### Fixed
- Исправлен баг: расчёт для точки входа больше не выполняется при некорректных данных (например, при алерте «Цена входа должна быть между TP и SL»). Для невалидной строки расчётные ячейки (средняя цена, объём, % до SL/TP, PnL, R/R) остаются пустыми; учитываются только «Цена входа» и «Сумма USDT». Невалидные строки не участвуют в сводке по позиции.

### Changed
- Сводка по позиции считается только по заполненным и валидным точкам входа. При добавлении новой (ещё не заполненной) точки входа сводка продолжает показывать данные по уже заполненным входам и не обнуляется.
- Блок «Сводка по позиции» отображается при наличии хотя бы одной рассчитанной точки входа (цена и сумма заполнены, валидации пройдены).
- Добавлено отображение тикера в поле "Общий объем".

## [0.3.0] - 2026-02-18

### Changed
- Убраны точки входа по умолчанию: калькулятор запускается с пустым списком входов.
- Убраны значения стоп-лосса и тейк-профита по умолчанию, а также плейсхолдеры в полях ввода.
- Добавлена валидация данных при добавлении нового входа.
- Добавление нового входа разрешено только после заполнения всех обязательных полей текущего входа.
- Кнопка «Добавить вход» перенесена в более логичное место интерфейса.

### Fixed
- Исправлены различные баги.

## [0.2.0] - 2026-02-03

### Fixed
- Fix local build errors (TypeScript unused variable warnings)

### Added
- Add Vercel deployment configuration with SPA routing support

## [0.1.0] - 2026-02-03

### Added
- Первый MVP релиз Risk Calculator
- Поддержка Long и Short позиций
- Множественные точки входа
- Динамический расчет средней цены позиции
- Расчет частичных сценариев для каждого входа
- Расчет Risk/Reward ratio
- Расчет прибыли/убытка при достижении SL/TP
- Расчет процентов движения цены
- Пресеты для быстрого ввода размеров позиций
- Сортировка входов (исходный порядок, по возрастанию, по убыванию)
- Предупреждение о подозрительных значениях R/R
- Адаптивный дизайн для десктопа
- Темная тема интерфейса
- Unit тесты для основной логики
- Документация (README, USAGE, ARCHITECTURE)

### Technical
- Vue 3 с Composition API
- TypeScript для типобезопасности
- Vite для быстрой разработки
- Pinia для управления состоянием
- Tailwind CSS для стилизации
- Vitest для тестирования
